# Customer Churn Prediction

End-to-end ML pipeline for predicting customer churn using the Telco Customer Churn dataset.

🚧 **Work in progress** — project is being actively developed.

## Tech stack

- Python 3.12
- pandas, NumPy
- scikit-learn, XGBoost
- matplotlib, seaborn
- Jupyter Notebook

## Dataset

[Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn) from Kaggle.
